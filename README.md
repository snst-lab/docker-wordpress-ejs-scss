## 作業用コマンド一覧
>  作業開始 : サーバ再起動 & BrowserSync再起動 & ファイル監視開始
```bash
gulp
``` 
> サイト生成 :  テンプレートファイルのコピー & 配布ディレクトリの削除　& ファイルのビルド
```bash
gulp init
```
>  サーバ停止
```bash
gulp down
``` 
>  再コンパイル
```bash
gulp build
```
